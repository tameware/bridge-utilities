Prepare ACBLScore results for a BridgeWinners article



This is the page you see if you click on the "Scores" link for a pair. Open the page and ask your browser to View Source (Chrome, Edge) or Show Page Source (Safari). Then Select All, Copy, close the page, and paste the result.

You can past the lines that are generated directly into a BridgeWinners article.

You can test the process out with this sample ACBL Live URL.


The source code for this utility is available on Github.
